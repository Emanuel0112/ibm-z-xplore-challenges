# IBM Z Xplore e Desafios COBOL 🚀

Este repositório foi criado para concentrar meus estudos práticos no ecossistema Mainframe. Aqui reúno laboratórios oficiais da trilha **IBM Z Xplore** e desafios práticos de lógica desenvolvidos na linguagem COBOL.

---

### 📂 Estrutura do Repositório

- **`/exercicios-cobol`**: Algoritmos e desafios práticos desenvolvidos no VS Code para fixação da sintaxe, estruturas de dados (Paragraphs, Variables) e lógica em COBOL.
- **`/ibm-z-xplore`**: Anotações, laboratórios e resoluções das trilhas conceituais e práticas da IBM.

---

### 🛠️ Ferramentas Utilizadas nos Testes
- **VS Code** (com extensões para sintaxe COBOL)
- **Open Side / Ambiente de Aprendizado IBM Z**
- **Sistemas de Arquivos/Datasets (Sequential e VSAM)**
